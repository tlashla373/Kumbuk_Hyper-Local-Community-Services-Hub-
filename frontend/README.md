# KumbuK Frontend

React Native + Expo mobile applications for KumbuK service platform.

## Apps

- **consumer-app**: Consumer-facing mobile app for finding services
- **provider-app**: Service provider app for managing business

## Quick Start

```powershell
# Install dependencies
npm install

# Start consumer app
npx nx start consumer-app

# Start provider app  
npx nx start provider-app
```

## Development

- TypeScript + React Native
- Nx monorepo for code sharing
- Expo for mobile development
- WebSocket for real-time chat
